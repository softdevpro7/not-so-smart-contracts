# (Not So) Smart Contracts

This repository contains examples of common Ethereum smart contract vulnerabilities, including code from real smart contracts.

## Vulnerabilities

- Integer Overflow
- Missing Constructor
- Reentrancy
- Unchecked External Call
- Unprotected Function
- Wrong Interface
